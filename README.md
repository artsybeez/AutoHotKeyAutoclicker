# Bleta - AutoClicker Hotkey

A lightweight autoclicker for AutoHotkey, designed to automate clicking for the **Simulation Clicker** game at [neal.fun/clicker](https://neal.fun/clicker/).

## How to Use

1. Install **AutoHotkey** from [autohotkey.com](https://www.autohotkey.com/)
2. Double-click `bleta.ahk` to run the script
3. The script runs in the background (look for the `H` icon in your system tray)
4. Press **`J`** to toggle autoclicking

## Controls

| Key | Action |
|-----|--------|
| `J` | Toggle autoclick on/off |

- When **on**: Clicks every 50ms (20 clicks per second)
- When **off**: Stops clicking immediately

## Purpose

This autoclicker was made to automate cookie/item clicking in the Simulation Clicker game on neal.fun, helping you accumulate clicks faster without wearing out your mouse.
